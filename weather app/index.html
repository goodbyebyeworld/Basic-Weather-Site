Weather Time is an open-source weather application that allows you to search for any city and view real-time weather conditions including temperature, humidity, wind speed, and more. You can switch between Celsius, Fahrenheit, and Kelvin units, save your recent searches, and even use your current location to get instant weather data.

This project was built with HTML, CSS, and JavaScript using WeatherAPI as the data provider.
